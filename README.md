# Logger

A Logging library used by DDC Dev. 
It is based on a project originally started by [Yan Cherkinov](https://www.youtube.com/user/TheChernoProject).
It is avaliable to anyone that wishes to use it, as long as the copyright notice is included.

This project has been officially discontinued by DDC Dev. A simpler, adapted version can be found at [the Nash library](https://github.com/DDC-Development/Nash)
